# PDF AI Assistant

This project extracts text from a PDF and allows:

- Ask questions from PDF
- Generate summary

## Technologies Used
- Python
- PyPDF2
- NLTK

## How to Run

1. Install dependencies:
   pip install PyPDF2 nltk

2. Run the program:
   python app.py

## Features
- PDF text extraction
- Question answering
- Summary generation
